Which file type should I upload?
Log File
Use for auth.log, syslog, or any system log. Enables Log Analyzer + Timeline.
PCAP Network Capture
Use for .pcap or .pcapng files. Enables PCAP Analyzer.
Email File (.eml)
Use for .eml email files. Enables Phishing Detection. Export emails as .eml from Outlook/Thunderbird.
Image File
Use for .jpg, .png, .tiff etc. Enables Metadata Extractor (EXIF, GPS, camera info).
Document
Use for .pdf, .docx, .xlsx, .pptx. Enables Metadata Extractor (author, company, revision).
Other / Malware Sample
Use for .exe, .dll, .bat, scripts, or any unknown file. Enables Malware Scanner.
