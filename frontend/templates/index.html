How It Works

1
Create a Case & Upload Evidence
Go to Upload Evidence. Name your case, add your investigator name, select the file type (log, PCAP, email, image, document), and upload the file. The toolkit automatically hashes the file for integrity.
2
Run Analysis Modules
Open the Dashboard. Click any evidence row to select it, then click the analysis button — Analyze Log, PCAP Analyzer, Malware Scan, Extract Metadata, or Phishing Detection. Results appear instantly in tabs.
3
Generate a PDF Report
Scroll to the Forensic Report section on the Dashboard. Add your conclusion and click Generate PDF Report. A professional multi-section PDF is created with all case details, evidence, findings, and timeline.
What you can analyze

Analysis Modules

Log Analyzer
auth.log / system logs

Parses Linux auth.log files to detect failed login attempts, brute-force attacks, and suspicious IP addresses. Auto-identifies IPs exceeding the threshold.

PCAP Analyzer
.pcap / .pcapng files

Uses Scapy to analyze network captures. Extracts connections, DNS queries, flags suspicious IPs with high connection counts, and detects unusual ports.

Malware Scanner
Any file

Checks file extensions, known malware hashes, and optional YARA rules. Computes SHA-256 fingerprint for integrity tracking.

Metadata Extractor
Images, PDFs, Office docs
NEW

Extracts hidden metadata: EXIF (camera, GPS location) from images, author/dates from PDFs, and revision history from Office documents (.docx/.xlsx/.pptx).

Phishing Detector
.eml email files
NEW

Analyzes email headers (SPF/DKIM/DMARC), detects Reply-To spoofing, display name impersonation, IP-based URLs, phishing keywords, and dangerous attachments. Returns a 0–100 risk score.

PDF Report Generator
Any case

Generates a professional forensic report with 6 sections: case overview, evidence vault, detected attacks, suspicious IPs, attack timeline, and your conclusion.
